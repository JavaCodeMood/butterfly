组件列表

  • Uploader

    异步文件上传组件，支持ajax、iframe、flash三种上传方案,由明河（剑平）、紫英、飞绿完成，请勿使用v1.0和v1.1的代码。

    demo doc使用指南

  • Limiter

    字数统计组件，支持对kissy的富编辑器editor的统计，由明河（剑平）完成。

    demo doc

  • SpinBox

    数字增减器，由易敛完成。

    demo doc

  • Auth

    表单验证，由张挺完成。

    demo doc

  • checkbox

    模拟多选框，用于表单美化，由伯方完成。

    demo doc

  • radio

    模拟单选框，用于表单美化，由伯方完成。

    demo doc
